Overview

planetrock is a Python library for accessing the recently played tracks on Planet Rock radio. It comes with a simple program prock that lists a given number of tracks from the last hour of radio play. prock or programs written using the library can be used from within conky or similar systems for an up to date feed of what is playing on the radio.
